Backend calls


Packages

  • GET /getPackages: retrieve all packages from database.
  • POST /getPackageById: pass a packageID and retrieve all package information.
  • POST /getPackageByTrackingNumber: pass a tracking number and retrieve all package information.
  • POST /createPackage: insert a newly created package into the database.
  • PUT /changeStatusToPickUpByTn: pass a tracking number and change the status of a package to "PickUp".
  • PUT /changeStatusToDeliveryByTn: pass a tracking number and change the status of a package to "Delivery".
  • PUT /changeStatusToDeliveredByTn: pass a tracking number and change the status of a package to "Delivered".
  • POST /createReport: insert a newly created report into the database.

Couriers

  • GET /getCouriers: retrieve all couriers from database.
  • POST /getCourierById: pass a courierID and retrieve all package information.
  • POST /getCourierReports: pass a courierID and retrieve all reports from this courier.

Invoices

  • GET /getInvoices: retrieve all invoices from database.

Statistics

  • GET /countExpress: retrieve the amount of express packages from the database.
  • GET /countEconomy: retrieve the amount of economy packages from the database.
  • GET /countInternational: retrieve the amount of international packages from the database.
